## Descrição 

Desafio da Alura com Oracle chamado  Challenge ONE, uma aplicação que criptografa textos, assim você poderá trocar mensagens secretas com outras pessoas que saibam o segredo da criptografia utilizada.

Por exemplo:

````

"gato" => "gaitober"
gaitober" => "gato" 

````
